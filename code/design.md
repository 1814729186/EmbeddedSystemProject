## 游戏程序状态机设计

状态注册：

```c
state 0 -- 开始阶段
state 1 -- 棋色选择阶段
state 2 -- 已选择黑棋并等待对手机确认选择
state 3 -- 已选择白棋并等待对手机确认
state 4 -- 对手机选择黑棋并等待本机确认
state 5 -- 对手机选择白棋并等待本机确认
state 6 -- 本机执子等待落子
state 7 -- 对手机执子等待对手机通信
state 8 -- 本机取胜状态
state 9 -- 本机判负状态
```
